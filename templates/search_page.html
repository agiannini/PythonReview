This application will allow you to leave reviews of any book in the database. It dynamically generates pages allows the user to post reveiws to its database. It also displays some info from the goodreads api.

The searchbar will query a SQL database hosted on Heroku. The searches are protected from SQL injection with SQLAchelmy. The search will produce complete and partial matches for author, title, isbn, and year. It is not case-sensitive.

The username defaults to anonymous, but register or login to make your own account. The registration process hashes the passwords using the werkzeug.security plugin.

The app also contains a feature which outputs JSON data for the given book by ISBN. To see this just search http://162.243.172.246:5000/api/$ISBN where $ISBN is the books ISBN.
